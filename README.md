# Denko-Tetris

シングルプレイ / 対戦プレイ が楽しめる SolidJS + TailwindCSS 製テトリス。  
フルスクリーン対応 (ルート要素が画面全体に固定)  

## 操作方法

- プレイヤー1 (シングル or 左側 in 対戦)
  - 左移動: A
  - 右移動: D
  - 回転: W
  - 左回転: Q
  - 180度回転: X
  - ソフトドロップ: S
  - ハードドロップ: Space
  - ホールド: C

- プレイヤー2 (対戦のみ 右側)
  - 左移動: ←
  - 右移動: →
  - 回転: ↑
  - 左回転: Ctrl
  - ソフトドロップ: ↓
  - ハードドロップ: Enter
  - ホールド: Shift

- 共通  
  - ポーズ / 再開: P  
  - リセット: R  

## 起動方法

```bash
npm install
npm run dev
```
